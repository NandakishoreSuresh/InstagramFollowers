Instagram Unfollowers Tracker

A web application that helps users identify Instagram accounts that do not follow them back and track changes in their follower/following lists over time.

Features:

Login using Instagram username

View followers and following lists

Detect accounts that:

Don't follow you back

You don't follow back

Recently unfollowed you

Search and filter accounts

Export reports as CSV

Responsive design for desktop and mobile

User-friendly dashboard
